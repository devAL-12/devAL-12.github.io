SSVEP — The Science

        

          When your eyes view a flickering light at frequency f, your visual cortex generates electrical oscillations at exactly f Hz (and harmonics 2f, 3f).


          This is called the Steady-State Visual Evoked Potential. It is detectable at the Oz electrode (occipital lobe).


          FFT converts the raw EEG time-series into frequency spectrum. The peak with highest power = the target the patient is staring at.
        

      

      

        
Browser FFT Pipeline

        

          EEG electrode (Oz)

          ↓ analog → TL074

          ↓ USB serial (115200)

          Web Serial API → JS buffer

          ↓ 4-second window

          Bandpass filter (1–40 Hz)

          ↓ FFT (1024 points)

          Peak power at 8/10/12/15 Hz

          ↓ argmax → command

          Web Serial → Arduino → Motor
        

      

    

    

      
WHY BROWSER-ONLY WORKS

      

        
✓ No Python install
All FFT runs in JS using Web Audio API math — zero dependencies.

        
✓ No server
Everything is local. Works offline once the HTML is downloaded.

        
✓ Real-time
JS processes 256 samples/sec. FFT window refreshes every 2 seconds.

        
✓ Portable
Works on any laptop with Chrome/Edge. Carry the HTML on a USB stick.
